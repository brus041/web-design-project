The purpose of this project was to use Python APIs to gather weather data. The majority of our analysis consists of observing trends at or around the equator. We will be focusing on four main factors and how they vary with changes in latitude. It can be observed that as latitude increases we approach the north pole and when it decreases we approach the south pole. As a result, many trends arise with potential inverse relationships. This is because as the latitude increases from the equator, it gets colder, whereas as it decreases it gets colder if we were to go the opposite way. The data we collected was derived from the OpenWeather API, through a simple TRY EXCEPT structure in order to filter out catch any unwanted, invalid data. As part of our analysis we used python to manipulate Json's, followed by utilizing Matplotlib to visualize our results in an organized manner. This webpage will then be created to show those results with js,html,and CSS.
